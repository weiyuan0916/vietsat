🔐 Auth API - Xác thực

Base URL: /api/v1/auth

1. Đăng ký tài khoản

POST /api/v1/auth/register

Request Body

Tham sốKiểuBắt buộcMô tả
namestring✅Họ tên người dùng
emailstring✅Email (duy nhất)
passwordstring✅Mật khẩu (≥6 ký tự)
password_confirmationstring✅Xác nhận mật khẩu

Response (201)

{
  "status": true,
  "message": "Đăng ký thành công.",
  "data": {
    "user": { "id": 1, "name": "Nguyễn Văn A", "email": "user@example.com" },
    "token": "1|abcdef..."
  }
}

2. Đăng nhập

POST /api/v1/auth/login

Request Body

Tham sốKiểuBắt buộc
emailstring✅
passwordstring✅

Response (200)

{
  "status": true,
  "message": "Đăng nhập thành công.",
  "data": { "user": {...}, "token": "..." }
}

3. Đăng xuất

POST /api/v1/auth/logout Cần token

Mô tả

Đăng xuất và xóa token hiện tại. Token sẽ không còn valid sau khi gọi API này.

Headers

Authorization: Bearer {your_token}
Accept: application/json

Response thành công (200)

{
  "status": true,
  "message": "Đăng xuất thành công."
}

Response lỗi (401) - Token không hợp lệ

{
  "status": false,
  "message": "Unauthenticated."
}

Ví dụ sử dụng

// JavaScript
async function logout() {
  const token = localStorage.getItem('token');
  
  const response = await fetch('/api/v1/auth/logout', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Accept': 'application/json'
    }
  });
  
  if (response.ok) {
    localStorage.removeItem('token');
    window.location.href = '/login';
  }
}

// Flutter
Future logout() async {
  final token = await storage.read(key: 'token');
  
  await http.post(
    Uri.parse('https://your-domain.com/api/v1/auth/logout'),
    headers: {
      'Authorization': 'Bearer $token',
      'Accept': 'application/json'
    }
  );
  
  await storage.delete(key: 'token');
}

💻 PC Infos API - Thông tin máy

Base URL: /api/v1/pc-infos

1. Lưu thông tin máy

POST /api/v1/pc-infos

Request Body

{
  "host_name": "DESKTOP-ABC123",
  "user_name": "john_doe",
  "password": "encrypted_password",
  "local_ip_address": "192.168.1.100",
  "public_ip_address": "203.0.113.1"
}

Logic

  • Nếu public_ip == local_ip → Cập nhật record hiện có
  • Nếu public_ip != local_ip → Tạo record mới

2. Danh sách máy

GET /api/v1/pc-infos

Query Parameters

Tham sốKiểuMô tả
host_namestringLọc theo tên máy
user_namestringLọc theo tên user
sort_bystringTrường sắp xếp (default: created_at)
sort_directionstringasc/desc
per_pageintegerSố item/trang (default: 15)

3. Chi tiết máy

GET /api/v1/pc-infos/{pcInfo}

4. Xóa máy

DELETE /api/v1/pc-infos/{pcInfo}

5. Thống kê

GET /api/v1/pc-infos/statistics/overview

React Native

const api = async (endpoint, options = {}) => {
  const token = await AsyncStorage.getItem('token');
  return fetch(`https://your-domain.com/api/v1${endpoint}`, {
    ...options,
    headers: {
      'Content-Type': 'application/json',
      'Accept': 'application/json',
      ...(token ? { 'Authorization': `Bearer ${token}` } : {}),
    },
  });
};

// Đăng nhập
const login = async (email, password) => {
  const res = await api('/auth/login', {
    method: 'POST',
    body: JSON.stringify({ email, password }),
  });
  return res.json();
};

// Lấy danh sách đơn hàng
const orders = await api('/orders/my-orders');

Swift (iOS)

func login(email: String, password: String) async throws -> AuthResponse {
    let url = URL(string: "https://your-domain.com/api/v1/auth/login")!
    var request = URLRequest(url: url)
    request.httpMethod = "POST"
    request.setValue("application/json", forHTTPHeaderField: "Content-Type")
    
    let body = ["email": email, "password": password]
    request.httpBody = try JSONEncoder().encode(body)
    
    let (data, _) = try await URLSession.shared.data(for: request)
    return try JSONDecoder().decode(AuthResponse.self, from: data)
}
